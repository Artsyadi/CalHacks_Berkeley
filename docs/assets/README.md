# Deck images

Drop your screenshots here with these EXACT filenames. `deck.html` loads them
automatically; any missing file shows a labeled placeholder so the deck still works.

| Filename | What it should show | Used on |
|----------|---------------------|---------|
| `a1-roadmap.png` | ASI:One reply with the Mermaid roadmap **rendered** + clickable links | Slides 3 & 4 (hero) |
| `a2-terminals.png` | The 4 agent terminals running, mid-request | Slide 5 |
| `a3-agentverse.png` | Orchestrator's Agentverse profile page | Slide 7 |
| `a5-payment.png` | (optional) `payment_demo` terminal showing `PAYMENT COMPLETE ✅` | Slide 8 |

PNG or JPG both work (keep the `.png` name, or edit the `data-img` in deck.html).

Capture on Windows: `Win+Shift+S` (screenshot) · `Win+Alt+R` (screen record).
